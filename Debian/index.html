Debian 9 (stretch)

Based configuration

You can use Debian Sources List Generator. Modify the data in the /etc/apt/sources.list file by new data.

apt update && apt upgrade

apt install sudo

adduser [username] sudo

apt install firmware-misc-nonfree - wi-fi driver

apt search [package_name]) - search the needed packages

dpkg-reconfigure tzdata - change the timezone

Troubleshooting

If you get GnuPG errors like:

GPG error: [..] Release: The following signatures couldnâ€™t be verified because the public key is not available: NO_PUBKEY [..]

apt update 2>&1 | grep -o '[0-9A-Z]{16}$' | xargs - try auto update

egrep -v '^#|^$' /etc/apt/sources.list{,.d/*.list} - try to find bad repo and do rm manually

gpg --keyserver subkeys.pgp.net --recv KEY gpg --export --armor KEY | sudo apt-key add - add the key manually
